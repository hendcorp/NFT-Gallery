== Description ==
  
NFT Gallery is a WordPress plugin that helps you to add NFTs from OpenSea to your WordPress site. You can sync NFTs from OpenSea to your WordPress site and show it on any posts or pages. Powered by OpenSea API.

**HOW TO CONFIGURE IT?**
1. Login to your WordPress site and go to WP OpenSea.
2. Insert your OpenSea API Key.
3. Choose which type of NFTs that you would like to show, either from a collection or a single wallet address.
4. Put your collection slug or wallet address.
5. Specify the number of NFTs to show.
6. Click Save Settings.
7. Copy the shortcode to any post or page on your website.
  
== Installation ==
  
1. Upload the plugin folder to your /wp-content/plugins/ folder.
1. Go to the **Plugins** page and activate the plugin.
